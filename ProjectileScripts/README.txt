# How to make a projectile based weapon.

1. Add a folder to 'ReplicatedServerStorage' named 'Remotes'.
2. Insert a 'RemoteFunction' into 'Remotes' named 'FireAttack'
3. Insert a script into 'ServerScriptStorage' named 'FireAttackScript'
4. Complete a weapon model with valid 'Barrel' and 'Handle' parts and insert the weld script.
5. Insert a folder into your tool named 'Scripts' and insert a LocalScript into 'Scripts'.
6. Copy the code for 'MainScript' into this script.

7. Copy the code for 'FireAttackScript' into the script in ServerScriptStoarge.

And you should be done!
